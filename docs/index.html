Instalación y e configuración

1. Añadir Plugin

En el escritorio del Wordpress, en el menú abre Plugins y selecciona Añadir Nuevo y escribe Moloni.es.

2. Activar Plugin

A continuación, haz clic en Instalar ahora. Cuando finalize haz clic en el botón Activar.


3. Creación del Cliente API

Previamente tienes que tener creado un cliente de API con el ID y el Client Secret en el Área de Cliente. Para saber como crear un Cliente API accede a esta Guía.


4. Autentificación del Cliente API

Coloca los datos que previamente has creado en el Área Cliente de Moloni, el correspondiente ID y Client Secret.


5. Iniciar Sesión

A continuación puedes debes colocar los datos para acceder a la cuenta de tu empresa.

6. Autorización

Ahora deberás autorizar y dar permiso para acceder a tu cuenta Moloni.


7. Selecciona la empresa

Aqui tendrás que seleccionar la empresa que deseas.

8. Configuraciones

En el separador de Moloni, dentro encuentras varios submenús que son Pedidos, Configuraciones y Herramientas. Selecciona el de las Configuraciones y selecciona las opciones que pretendes y que datos se quedaran por defecto.


9. Crear Pedidos

Los pedidos aparecen en el menú inicial del plugin de Moloni para que sean creados.

Preguntas Frecuentes

  • labelComo crear un documento

    Cuando el pedido ya ha sido pagado, va aparecer en una lista de documentos que tiene pendientes para enviar para Moloni. Desde ahí, haz clic para crear el documento, añadir los productos, y el respectivo cliente en tu cuenta Moloni.

  • labelError al crear el documento

    En la configuración puedes consultar los registros con detalles sobre los errores. Cualquier duda no dude en contactar con nosotros.

  • label¿Por qué tengo que colocar el motivo de la exención si los productos no están exentos?

    El motivo de exención seleccionado es un motivo de exención a ser usada por defecto. Si los productos tiene un impuesto asociado en el pedido, el motivo de exención no va a ser usado.

  • labelComo asociar gastos de envío

    Los gastos de envío son añadidos de forma automática al documento como un producto. Por eso, tendrás que verificar que está configurado de forma correcta y con el IVA correspondiente.

  • labelLos valores finales no están correctos.

    Este problema puede ocurrir cuando se redondea, y si se realiza, el documento no se cierra y te presenta un error informandote de lo sucedido. Sin embargo, puedes encontrar el documento creado en Moloni en “borrador” y puedes editar el documento sin ningún problema.
